# Task-Scheduler-Angular

Schedule tasks to be completed by a specific date.

A Task Scheduling application built by Jeremy Price to demonstrate skills in Angular, NgRx and Angular Reactive Forms.

## Features

- Add, edit, and remove tasks
- Mark tasks as complete or reopen completed tasks
- View all tasks sorted by due date
- Form validation using Angular Reactive Forms
- State management with NgRx
- Responsive design

## Technology Stack

- **Framework**: Angular 18
- **State Management**: NgRx 17
- **Forms**: Angular Reactive Forms
- **Styling**: CSS3
- **Build Tool**: Angular CLI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm start

# Navigate to http://localhost:4200/
